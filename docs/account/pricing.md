## Créditos

- 1 crédito = 1 minuto de vídeo processado
- Créditos são renovados mensalmente
- Planos pagos incluem bônus de boas-vindas

## Upgrade

1. Acesse **Configurações** → **Plano atual**
2. Clique em **Fazer upgrade**
3. Selecione o plano desejado
4. Complete o pagamento via Stripe

## Enterprise

Para planos Enterprise, entre em contato via WhatsApp ou email para negociação personalizada com:

- GPU dedicada
- API access completa
- Gestão de equipe
- SLA garantido
- Suporte 24/7 prioritário
